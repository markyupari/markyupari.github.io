Eyes movement animation

This page shows how to combine CSS with JavaScript and DOM manipulation to create an animation of eyes following the cursor movement.

This can be done by getting the event of "onmousemove" and converting it to the position of the 'eyes'. Just hover the mouse around the page to see how your movement is been tracked.

Real Time Bus Tracker

This project is an example of how to show a map with a Real-time bus tracker.

As the bus location changes, it's updated on the map and auto-centered. The marker is customized to show an image of a bus.

The location refresh rate is 15 seconds. To begin the bus tracking, click on the button "Start Bus Tracker."
